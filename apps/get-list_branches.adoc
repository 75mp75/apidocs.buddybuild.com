== Definition

[.definition.placeholder]
Placeholder for the definition; auto-generated from JSON above.

Use this method to retrieve a list of branches associated with the
specified `app_id`.

Deleted branches are not included in the response by default. To include
deleted branches in the response set the `includeDeleted` query
parameter to `true`.
